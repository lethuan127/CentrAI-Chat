# Skills (Markdown)

Optional **human- and tool-facing** skill descriptions for this repo (workflow checklists, prompts, and conventions). These are **not** loaded by the Go runtime today—the runtime skill pipeline is described in [docs/7. skills.md](../../docs/7.%20skills.md).

Use the same shape as Cursor **Agent Skills**: YAML front matter + Markdown body.

| Field | Purpose |
|-------|---------|
| `name` | Short id |
| `description` | When to use this skill |
