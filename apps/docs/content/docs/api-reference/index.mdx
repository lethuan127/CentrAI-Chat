CentrAI-Chat exposes a RESTful API at `/api/v1`. All endpoints follow the consistent envelope format:

```json
{
  "data": { ... },
  "error": null,
  "meta": { "total": 42, "page": 1, "limit": 20, "totalPages": 3 }
}
```

## Interactive Docs

When the API is running, the full interactive Swagger UI is available at:

**[http://localhost:4000/api/docs](http://localhost:4000/api/docs)**

You can authenticate with a JWT token and use "Try it out" to test endpoints directly.

## OpenAPI Spec

The machine-readable OpenAPI 3.0 JSON spec is available at:

**[http://localhost:4000/api/docs-json](http://localhost:4000/api/docs-json)**

Use this to:
- Import into Postman, Bruno, Insomnia, or any API client
- Generate client SDKs in any language
- Validate the API with `swagger-cli validate`

## Authentication

Most endpoints require a Bearer JWT token in the `Authorization` header:

```
Authorization: Bearer eyJhbGciOiJIUzI1NiIs...
```

Obtain a token via `POST /api/v1/auth/login`. The access token expires after 15 minutes by default; use `POST /api/v1/auth/refresh` with the refresh token to get a new one.
