CentrAI-Chat is an open-source platform that gives teams a unified chat interface for AI conversations. **Admins** configure LLM providers and publish agents; **end users** sign in, pick an agent or model, and chat.

## Key Features

- **Multi-provider** вҖ” OpenAI, Anthropic, Google Gemini, Ollama, and any OpenAI-compatible endpoint
- **Agent builder** вҖ” Create agents with system prompts, model bindings, versioning, and publish workflow
- **Streaming chat** вҖ” Real-time token-by-token responses via SSE
- **Conversation branching** вҖ” Fork from any assistant message and explore alternative continuations
- **Thinking & tool use** вҖ” Reasoning blocks and tool call/result steps are persisted and surfaced in the UI
- **Admin dashboard** вҖ” User management, analytics, audit log, system settings
- **Self-hostable** вҖ” One-command Docker Compose deployment
- **RBAC** вҖ” Admin, Developer, and User roles with fine-grained permissions
